# 🍔 Apptizer

Welcome to **Apptizer**, a modern iOS application built with **SwiftUI**.

## 📱 Features

- Browse delicious appetizers
- View detailed food information
- Add items to your order
- Manage your account
- Clean and modern UI
- Built using MVVM architecture

## 🛠️ Technologies

- Swift
- SwiftUI
- MVVM
- Async/Await
- SF Symbols
